Before Interview

ถามผมก่อนเริ่มสัมภาษณ์

ปัญหาที่เจอบ่อยที่สุดในงาน UX/Product Design คืออะไร และคุณรับมือยังไง?

ปัญหาที่เจอบ่อยที่สุดคือความไม่ชัดเจนที่สะสมอยู่ในระบบ ทั้ง requirement ที่กระจัดกระจาย ownership ที่ไม่แน่นอน และ flow ที่แต่ละทีมเข้าใจไม่ตรงกัน ผมเริ่มจากสัมภาษณ์คนทำงานจริง map ขั้นตอนและข้อจำกัด แล้วเปลี่ยนสิ่งที่คลุมเครือให้กลายเป็น workflow, screen logic และ decision point ที่ทีมออกแบบ พัฒนา และ operation ใช้ร่วมกันได้

คุณคิดว่าผู้นำที่ดีเป็นยังไง?

ผู้นำที่ดีคือคนที่ทำให้ทีมเห็นบริบทเดียวกันและตัดสินใจได้ชัดขึ้น ไม่ใช่แค่สั่งงานหรือมีคำตอบเร็วที่สุด แต่ต้องช่วยลด noise แยกเรื่องสำคัญออกจากเรื่องเร่งด่วน และสร้างพื้นที่ให้คนอื่นทำงานได้ดีขึ้น ผมให้ค่าน้ำหนักกับความชัดเจน ความรับผิดชอบ และการทำให้ทีมเดินต่อได้โดยไม่ต้องเดา

ทำไมถึงเลือกทำงานสาย UX/Product/System Design?

ผมเลือกสายนี้เพราะชอบเปลี่ยนปัญหาที่ซับซ้อน กระจัดกระจาย และพูดยาก ให้กลายเป็นโครงสร้างที่คนอื่นใช้งานต่อได้จริง งานที่ดีสำหรับผมไม่ใช่แค่หน้าจอสวย แต่คือระบบที่ช่วยให้คนเข้าใจงานของตัวเอง ลดการทำงานนอกระบบ และทำให้ทีมส่งต่องานกันได้อย่างมั่นใจขึ้น
